This is a job recommendation platform that runs on streamlit (open-source Python framework designed to help data scientists and machine learning engineers create interactive web applications with minimal effort and no frontend development experience).


if you want to use this project then keep in mind to change the app.py according to your preferences.

The generate_data.py is a python program that i written to randomly generate the data in the form of csv files.
    You can change the number of roles and the skill pools according to your preferences.

The candidates.csv file is not being used but can be used if needed to act as a webpage that give recommendations based on the pre determined candidates.
